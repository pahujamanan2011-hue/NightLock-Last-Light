# NightLock: Last Light

Retro top-down horror survival game.

## Controls
- WASD / Arrows = Move
- Mouse = Aim
- Left Click = Shoot
- 1 = Pistol
- 2 = Shotgun

## Setup
1. Put all PNGs into assets/images
2. Put MP3 files into assets/audio
3. Run index.html

Optional:
python server.py
